### Installing

* Fork this repo.
  [repo](https://github.com/Saxenaa218/coin-dash)
* Clone the repo using
  ```
    git clone https://github.com/Saxenaa218/coin-dash
  ```
* Change directory
  ```
    cd coin-dash
  ```
* Install dependencies
  ```
    npm install
  ```
* Run app
  ```
    npm start
  ```
